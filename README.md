# Retail Product Performance Analysis

This project analyzes retail sales data to understand product performance and revenue distribution across categories.

## Tools Used
- Python
- Pandas
- Matplotlib

## Project Objectives
- Identify top-selling products
- Analyze revenue contribution by product
- Compare category performance
- Visualize product revenue trends

## Analysis Steps
1. Data exploration
2. Revenue calculation using price and units sold
3. Grouping data by product and category
4. Ranking products by total revenue
5. Visualizing product revenue using bar charts

## Key Insights
- Phones generated the highest revenue among all products.
- Laptops are the second highest revenue-generating product.
- Electronics category significantly outperforms accessories.
- Accessories contribute smaller portions of overall revenue.
